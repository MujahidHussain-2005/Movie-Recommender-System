# 🎬 Movie Recommender System

A simple Movie Recommender System built using Machine Learning that suggests similar movies based on movie metadata.
The system helps users discover movies by analyzing similarities between different films.

---

## 🚀 Features

* Movie recommendations based on similarity
* Content-based filtering
* Fast similarity search using cosine similarity
* Interactive web interface using Streamlit

---

## 🛠 Tech Stack

* Python
* Pandas
* NumPy
* Scikit-Learn
* Streamlit

---

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/MujahidHussain-2005/Movie-Recommender-System.git
cd Movie-Recommender-System
```

Run the Streamlit application

```bash
streamlit run app.py
```

---

## 📊 How It Works

1. Movie metadata is cleaned and processed.
2. Important features such as genres, keywords, cast and overview are combined.
3. Text data is converted into vectors using TF-IDF or CountVectorizer.
4. Cosine similarity is used to compute similarity between movies.
5. The system returns the most similar movies as recommendations.
